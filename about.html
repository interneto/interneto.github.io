About

Interneto is a hobby project created by David Alonso. Today is an alpha state but the idea is to evolve and improve even the app Interneto-mannger and the directory Interneto-links. This project is totally free so for now it only would be a local app, but maybe in the future could be an online manager app. That's depend on you because this cost money to pay the server and the domain.

Interneto is the combination of: In + ter + net + o = Inside of the tera (1 trillion) networks of the closed circular connection (o).
